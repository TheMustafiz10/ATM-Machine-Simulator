# ATM Machine Simulator

This ATM Machine Simulator is a low-level system programming project that simulates the core functionality of an ATM using [Assembly Language](https://en.wikipedia.org/wiki/Assembly_language). The simulator demonstrates essential banking operations, including PIN-based authentication, balance inquiries, deposits, withdrawals, and transaction history, providing a realistic experience of how ATMs operate at the hardware and software level.

---

### ✨ Features

* **PIN Authentication**: Secure login with a limited number of attempts.
* **Error Handling**: Displays messages for incorrect PIN entries and redirects to the main menu.
* **Balance Inquiry**: View the current account balance at any time.
* **Cash Withdrawal**: Deducts a user-specified amount and updates the balance.
* **Cash Deposit**: Adds a user-specified amount to the account balance.
* **Transaction History**: Displays a record of recent deposits and withdrawals.
* **Logout Option**: Return to the main menu after any operation.

---This project is an excellent learning tool for understanding low-level programming concepts, memory management, and the step-by-step logic behind real-world banking systems.


### 🔧 Installation & Usage

1. **Requirements**

   * An x86 emulator like [EMU8086](http://www.emu8086.com/) or DOSBox
   * Basic knowledge of x86 Assembly

2. **Running the Simulator**

   * Open the `.asm` file in your emulator.
   * Compile and run the program.
   * Follow the on-screen prompts to interact with the ATM system.

3. **Controls**

   * Enter a 4-digit PIN to log in.
   * Choose an action from the menu (e.g., balance check, deposit, withdraw).
   * Use the logout option to return to the main menu safely.
